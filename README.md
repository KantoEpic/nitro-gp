[Nitro GP](https://wiki.tockdom.com/wiki/Nitro_GP) is a custom track distribution created by [Narceja](https://wiki.tockdom.com/wiki/User:Narceja). It adds many custom cups and retro cups, as well as a new UI and features.

This is the source code for the distribution using the [Pulsar](https://github.com/MelgMKW/Pulsar) engine by [MelgMKW](https://github.com/MelgMKW), based on the work of [Kamek](https://github.com/Treeki/Kamek/tree/master) by [Treeki](https://github.com/Treeki). Most of the edited code comes from [Yosh's Track Pack DX's source code](https://github.com/YoshTaku/ytpdx3), which itself uses the [OptPack source code](https://github.com/Optllizer/OptPack) as a base.
